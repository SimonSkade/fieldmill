\title{Die Feldm\"uhle}
\author{Simon Skade}
\begin{document}
\maketitle
\section{Einf\"uhrung}
Die Feldm\"uhle ist ein Messger\"at zur
Bestimmung der elektrischen Feldst\"arke.
\subsection{Anwendungsm\"öglichkeiten}
Innerhalb eines Gewitters ist das
elektrische Feld sehr viel h\"öher als
normalerweise. Daher werden Feldm\"uhlen
verwendet, um Risiken von Gewittern und
Blitzeinschl\"agen zu berechnen.
Beispielsweise findet diese Technik bei
Raketenstarts Anwendung.
Weiterhin wird es zur Bestimmung des
elektrischen Feldes in Laboratorien zur
\"uberpr\"ufung der Versuchsbedingungen
verwendet.
Es ist auch nur ein interessantes
Messger\"at.
\section{Dokumentation}
\subsection{Aufbau}
Die Feldm\"uhle besteht aus einer
Kupferplatine, die vier wesentliche
Segmente hat. Diese Segmente sind
gleichm\"aßige Ausschnitte eines Kreises,
von denen jedes mit dem
gegen\"uberliegendem Segment verbunden
ist.

Die Feldm ̈uhle ist ein Messger ̈at zur Bestimmung der elektrischen Feldst ̈arke.
1.1 Anwendungsm ̈glichkeiten
Innerhalb eines Gewitters ist das elektrische Feld sehr viel h ̈her als normalerweise. Daher werden

2.1 Aufbau
Die Feldm ̈uhle besteht aus einer Kupferplatine, die vier wesentliche Segmente hat. Diese Segmente sind
gleichm ̈aige Ausschnitte eines Kreises, von denen jedes mit dem gegen ̈uberliegendem Segment verbunden

drehen. Der Elektromotor ist an eine Stromquelle angeschlossen und kann durch einen Schalter/Transitor
eingeschaltet werden. An zwei nebeneinanderliegenden Segmenten ist ein 1M-Ohm Wiederstand zwischen
der Vorder- und R ̈uckseite der Platine angeschlossen.
2.2 Funktionsweise
Die Kupfersegmente gewinnen Ladung aus dem elektrischen Feld in der Luft. Wenn der Rotor ̈uber
ein Segment kommt, absorbiert er die Ladungen, so dass das Segment weniger wird und umgekehrt,
wenn sich der Rotor von einem Segment entfernt. Dadurch ergibt sich eine Potentialdifferenz, die zu
Spannung und Strom f ̈uhrt. ̈uber den 1M Ohm Widerstand messen wir die Spannung. Wir k ̈nnen die
Empfindlichkeit berechnen (Sie sehen, wie in Anweisungen -¿ Programm -¿ Grenzwert finden), um das
tats ̈achliche elektrische Feld aus unserem Spannungsausgang herauszufinden.
In unserem Fall geht die Ausgangsspannung an einen Analog-Digital-Wandler (wir benutzen den ADS1115)
und dann an den Raspberry Pi (Null W). Wenn eine obere Grenze ber ̈uhrt wird, speichert der PI die

def convergetangentialline(f, a, x1, x2, nbrofplots=50, epsilon=.1):
"""
Function to make a tangential line converge
"""
clrs = rainbow(nbrofplots)
k = 0
h = epsilon
p = plot(f, x, x1, x2)
while k < nbrofplots:
tangent(x) = fdash(f, a, h) * x + f(a) - fdash(f, a, h) * a
p += plot(tangent(x), x, x1, x2, color=clrs[k])
h += epsilon
k += 1
return p
